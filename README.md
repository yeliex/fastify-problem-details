# fastify-problem-details monorepo

This repository is a pnpm workspace with two packages:

- `@yeliex/problem-details`: RFC 9457 Problem Details core model and types.
- `@yeliex/fastify-problem-details`: Fastify integration and HTTP error factories.

## Development

```sh
pnpm install
pnpm test
pnpm build
```

## Packages

- `packages/problem-details`
- `packages/fastify-problem-details`
